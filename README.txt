Sticker Studio by UX Live — мини-редактор телеграм-стикеров
============================================================

Превращает ваше видео в готовый видеостикер для Telegram:
WebM (VP9), не больше 256 КБ, без звука, одна сторона 512px —
и с обходом лимита «не длиннее 3 секунд» (можно до 6 секунд).

КАК ПОЛЬЗОВАТЬСЯ
1. Запустите StickerStudio.exe и перетащите в окно видео:
   - .mov или .webm с альфа-каналом (прозрачность сохранится);
   - обычный .mp4 — если фон однотонный (зелёный), его можно удалить.
2. В редакторе:
   - Play — зацикленный просмотр без звука, как будет в Telegram;
   - таймлайн: синяя рамка — какой кусок попадёт в стикер
     (от 0,5 до 6 секунд, двигайте рамку и её края);
   - Crop 1:1 — выбор квадратной зоны стикера. Если видео больше
     512px, кроп обязателен (программа подскажет);
   - Удалить фон — хромакей: пипеткой укажите цвет фона,
     Gain — сила вырезания, Shrink/Grow — поджать/расширить края;
   - Отменить — откат последнего действия.
3. Экспорт — рядом с исходником появится имя_sticker.webm.
   Его и загружайте в Telegram (@stickers).

ПРОЗРАЧНОСТЬ
Была альфа на входе — будет на выходе. Удалили зелёный фон —
стикер тоже будет прозрачным. Обычный mp4 без удаления фона —
стикер с фоном (это нормально для «квадратных» стикеров).

ТРЕБОВАНИЯ
Windows 10/11, ничего устанавливать не нужно.
- StickerStudio.exe — рядом должен лежать ffmpeg.exe (уже в папке);
- StickerStudio-Standalone.exe (~36 МБ) — всё в одном файле,
  удобно пересылать; при первом запуске пару секунд распаковывается.
